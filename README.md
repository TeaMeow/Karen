

# Karen

可憐（カレン）是一個基於 PHP 的多國語言套件，主要以儲存為語言檔案為主並且支援評分系統，

你可以將可憐用於小型網站或者是大規模網站。

&nbsp;

# 特色

1. 以關鍵字為索引

2. 簡潔的使用方式

3. 支援評分優先（可選）

4. 將語系儲存為檔案

5. 線上免安裝編輯界面
